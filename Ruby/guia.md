# Sistema de Agendamento com Restrições (T3)

## Como Instalar o Ruby
* **Windows**: Baixar o RubyInstaller com Devkit no site oficial.
* **Linux/macOS**: Utilizar o gerenciador `rbenv` ou o gerenciador de pacotes nativo (`sudo apt install ruby`).

## Como Executar o Programa
1. Abra o terminal na pasta do projeto.
2. Execute o comando:
   ruby main.rb
